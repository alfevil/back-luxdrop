Серверная часть интернет-магазина LUXDROP, обеспечивающая бизнес-логику, работу с базой данных и безопасность пользовательских данных.

## 🛠 Технологический стек
* **Runtime:** Node.js 20.
* **Framework:** Express.js 4.
* **Database:** PostgreSQL 15 (развернута на Supabase).
* **Auth:** JWT (JSON Web Tokens) + bcrypt (хеширование паролей, 12 раундов).
* **Infrastructure:** Railway (PaaS).

## 🔑 Ключевой функционал
* **Безопасность:** Защита от SQL-инъекций через параметризованные запросы и настроенная CORS-политика.
* **REST API:** 12 эндпоинтов для управления продуктами, категориями, корзиной и заказами.
* **Middlewares:** Проверка авторизации для защищенных роутов (профиль, заказы).
* **База данных:** Реляционная структура из 7 таблиц.

## 📡 Основные эндпоинты
* `POST /api/auth/login` — Аутентификация и выдача токена.
* `GET /api/products` — Получение списка товаров с фильтрацией.
* `GET /api/cart` — Работа с корзиной (требуется JWT).
* `POST /api/orders` — Оформление заказа.
